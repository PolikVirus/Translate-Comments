# Translate Comments

This mod does exactly what you would expect... 
it translates comments.

Unfortunately, in some languages, special marks on letters (diacritics) such as Ā or Ñ will appear as plain A or N. Thats because the Geometry Dash comment font does not support these special characters.

The mod supports translating to:
 * English
 * Spanish
 * German
 * French
 * Swedish
 * Latvian
 * Estonian
 * Lithuanian
 * Portuguese
 * Polish
 * Italian
 * Dutch
 * Turkish
 * Romanian
 * Finnish
 * Danish
 * Norwegian
